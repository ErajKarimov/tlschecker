# TLSCHECKER
- Tool for scanning domains and checking there experied day
## How use it
- Python script
```sudo nice --20 taskset -c 0-$(nproc) python3 script.py```
- Goland script
```sudo nice --20 taskset -c 0-$(nproc) go run script.go```
- Bash script
```sudo nice --20 taskset -c 0-$(nproc) ./script.sh```
